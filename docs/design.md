# Apartment Management System - Design Document

## Overview
This project is an Apartment Management System developed using Java. It helps manage resident details.

## Classes Used
Resident, Tenant, Owner, Apartment, Main, ApartmentGui, Exception

## Relationships
Tenant and Owner inherit from Resident.

## OOP Concepts
Encapsulation, Inheritance, Polymorphism, Abstraction, Exception Handling

## Data Handling
ArrayList is used to store data temporarily.

## Future Improvements
Add database and better GUI
